# springboot-learn

本项目主要是为了学习springboot，会集成项目开发中常用的框架

目前springboot集成了 spring-security dubbo

dubbo : 分别实现了客户端和服务端，运行需要自主配置zookeeper

spring-security : 实现了authserver和resourceserver
